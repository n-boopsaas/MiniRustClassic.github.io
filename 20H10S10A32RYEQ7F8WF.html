Чистый ванильный Rust

Никаких модов, которые ломают баланс. Никаких китов при заходе. Только хардкор с нуля — как задумали разработчики. Бегаешь с камнем, пока не найдешь нормальный лут — тут так.

MiniRustClassic

Сервер для тех, кто скучает по старым временам в Rust. Без лишних накруток, с нормальным администрированием и без всей этой современной фигни. Админ всегда на связи, по читерам прилетает быстро.

Активная поддержка

Если кто-то читерит — пиши, админ забанит. Вопросы по серверу тоже решаются быстро. Иногда проводятся мелкие ивенты для своих, чтобы разнообразить геймплей.

Сервер крутится на нормальном железе, пинг низкий. Карта обновляется раз в пару недель, чтобы не застаиваться, но без драм. В общем, залетай, проверим, насколько ты готов к реальному выживанию.
